description list with 5 HTML tags

  1. ol : An ordered list.
  2. ul : An unordered list.
  3. li : List item. If used with ol tag, list items will be marked with bullets. If used with ul tag, the list items will be marked with numbers by default
  4. strong : a string of highly emphasized, much more important text. Screen readers will change their voice for this text.
  5. q : a small quotation embedded within other content. You become what you think.
